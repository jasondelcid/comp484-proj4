# Timed Typing Test App

A typing speed test web application built with HTML, CSS, and JavaScript for COMP 484.

## Features

- Live typing timer
- Real-time input validation
- Dynamic border color feedback
- Words Per Minute (WPM) tracker
- Error counter
- Random typing paragraphs
- Top 3 fastest scores saved using Local Storage
- Reset functionality

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Download or clone the repository
2. Open index.html in your browser
